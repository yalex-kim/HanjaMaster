# 한자 마스터 (Hanja Master) 🐉

초등 저학년 아이가 게임을 하듯 즐겁게 **한자능력검정시험 6급**을 준비할 수 있는 웹 기반 학습 앱

## 주요 기능

### 4가지 게임 모드

| 모드 | 설명 |
|------|------|
| 📝 **한자 퀴즈** | 4지선다 퀴즈 (한자→뜻음, 뜻음→한자, 한자→음) |
| 🃏 **카드 매칭** | 한자↔뜻음 짝 맞추기 (3D 카드 뒤집기) |
| ✍️ **쓰기 연습** | Canvas 기반 자유 필기 + 힌트 가이드 |
| 📖 **한자 복습** | 플래시카드 (알아요/모르겠어요 분류) |

### 보상 시스템
- XP 경험치 & 레벨업
- 연속 정답 스트릭 🔥
- 하트 시스템 ❤️

### 학습 범위
한국어문회 전국한자능력검정시험 6급 배정한자 **300자** (8급~6급)

| 급수 | 한자 수 |
|------|--------|
| 8급 | 50자 |
| 준7급 | 50자 |
| 7급 | 50자 |
| 준6급 | 75자 |
| 6급 | 75자 |

## 기술 스택

| 구분 | 기술 |
|------|------|
| 프레임워크 | React 18 |
| 빌드 | Vite 5 |
| 스타일링 | CSS-in-JS (Inline Styles) |
| 사운드 | Web Audio API |
| 쓰기 | HTML5 Canvas |
| 데이터 저장 | localStorage |

## 시작하기

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```

## 프로젝트 구조

```
src/
├── data/           # 한자 데이터 (8급~6급, 300자)
├── hooks/          # 커스텀 훅 (게임상태, 사운드, 캔버스, 스토리지)
├── components/     # 공통 컴포넌트 (TopBar, XPBar, FlipCard 등)
├── screens/        # 화면 (Home, Quiz, Match, Write, Review)
├── styles/         # 테마, 글로벌 스타일
├── utils/          # 유틸리티 (셔플, 퀴즈 생성)
├── App.jsx         # 메인 앱
└── main.jsx        # 엔트리포인트
```
