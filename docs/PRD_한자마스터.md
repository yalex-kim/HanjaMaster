## 1. 제품 개요

### 1.1 제품명
**한자 마스터 (Hanja Master) 🐉**

### 1.2 제품 비전
초등 저학년 아이가 게임을 하듯 즐겁게 한자능력검정시험 6급을 준비할 수 있는 웹 기반 학습 앱

### 1.3 핵심 가치
- **재미**: 게임 요소(퀴즈, 카드매칭, 보상)를 통한 자발적 학습 동기 부여
- **효과**: 시험 출제 유형에 맞춘 실전 대비 학습
- **접근성**: 별도 설치 없이 브라우저에서 바로 사용

### 1.4 배경 및 목적
한국어문회 주관 전국한자능력검정시험 6급은 초등학생 권장 급수로, 읽기 300자 / 쓰기 150자를 다룹니다. 기존 한자 학습은 단순 암기 위주로 저학년 아이들의 흥미를 끌기 어렵습니다. 게임식 접근으로 학습 지속성과 효과를 높이고자 합니다.

## 2. 타겟 사용자

### 2.1 주요 사용자 (Primary)
| 항목 | 내용 |
|------|------|
| **연령** | 초등 1~2학년 (만 7~8세) |

| **특성** | 짧은 집중 시간, 즉각적 보상 선호, 터치 인터페이스에 익숙 |
| **사용 환경** | 스마트폰/태블릿 브라우저 (보호자 기기) |

### 2.2 보조 사용자 (Secondary)
| 항목 | 내용 |
|------|------|
| **역할** | 학부모 |
| **니즈** | 아이의 학습 진도 확인, 학습 시간 관리 |
| **향후** | 학부모 대시보드 (Phase 4) |

> 한자에 관심은 있지만 쓰기 반복 학습이 지루해서 중간에 포기하곤 합니다. 스마트폰 게임을 좋아하고 스티커 모으기, 레벨업에 큰 동기를 느낍니다. 하루 15~20분 학습이 적당합니다.

## 3. 제품 범위

### 3.1 시험 범위
한국어문회 전국한자능력검정시험 6급 배정한자:

| 급수 | 신규 한자 수 | 누적 한자 수 | 비고 |
|------|-------------|-------------|------|
| 8급 | 50자 | 50자 | 기초 입문 |
| 준7급 | 50자 | 100자 | |
| 7급 | 50자 | 150자 | |
| 준6급 | 75자 | 225자 | 쓰기 시작 |
| **6급** | **75자** | **300자** | **읽기 300자, 쓰기 150자** |

### 3.2 시험 출제 유형 (6급)
1. 한자의 훈음 (뜻과 음) 쓰기
2. 훈음에 해당하는 한자 고르기
3. 한자어의 독음 쓰기
4. 독음에 해당하는 한자어 고르기
5. 동음이의어 구별
6. 반대어 / 유의어
7. 한자 쓰기 (7급 범위 150자)
8. 총획수 / 부수

## 5. 비기능 요구사항

### 5.1 성능
| 항목 | 기준 |
|------|------|
| 초기 로딩 | 3초 이내 (Wi-Fi 환경) |
| 화면 전환 | 300ms 이내 |
| 애니메이션 | 60fps 유지 |
| 번들 크기 | 500KB 이하 (초기 로드) |

### 5.2 호환성
| 항목 | 기준 |
|------|------|
| 브라우저 | Chrome 90+, Safari 15+, Samsung Internet 15+ |
| 디바이스 | iOS 15+, Android 10+ 스마트폰/태블릿 |
| 해상도 | 320px ~ 430px 너비 최적화, 768px까지 대응 |

### 5.3 접근성
- 최소 폰트 크기 14px (본문), 한자 표시 최소 32px
- 색상 대비 비율 4.5:1 이상
- 터치 타겟 최소 44x44px

### 5.4 데이터/보안
- 개인정보 수집 없음 (로그인 불필요)
- 학습 데이터는 브라우저 localStorage에만 저장
- 향후 클라우드 동기화 시 암호화 적용

## 8. 제약사항 및 가정

### 제약사항
- 웹앱 전용 (네이티브 앱 미지원)
- 오프라인 모드 미지원 (Phase 1)
- 한자 필기 자동 인식 미지원 (Phase 1)
- 한국어문회 배정한자 기준 (타 기관 급수와 상이할 수 있음)

### 가정
- 사용자는 보호자의 기기(스마트폰/태블릿)를 사용
- 인터넷 연결 환경에서 주로 사용
- 하루 15~20분 학습 시간 확보 가능
- 보호자가 초기 접속 및 설정을 도와줌
